🎬 게임 스토리보드
장면 1: 게임 시작 화면
화면 요소:

게임 제목: “🌍 수도 퀴즈 챌린지”

간단한 설명: “3개 국가의 수도를 맞춰보세요! 정답 시 쿠폰을 드려요!”

시작 버튼: “▶ 게임 시작”

버튼 액션:

버튼을 누르면 장면 2로 이동

장면 2: 문제 1 - 프랑스
질문: “🇫🇷 프랑스의 수도는?”

입력 필드: 객관식으로 4개의 보기를 제시하라

제출 버튼: “제출하기”

기능:

입력 후 제출 버튼을 누르면 정답 여부 체크

정답 체크 후 장면 3으로 이동

장면 3: 문제 2 - 브라질
질문: “🇧🇷 브라질의 수도는?”

입력 필드: 객관식으로 4개의 보기를 제시하라

제출 버튼: “제출하기”

기능:

동일한 구조, 입력 후 장면 4로 이동

장면 4: 문제 3 - 호주
질문: “🇦🇺 호주의 수도는?”

입력 필드: 객관식으로 4개의 보기를 제시하라

제출 버튼: “제출하기”

기능:

정답 체크 후 결과 장면으로 이동

장면 5: 결과 화면
조건 분기:

3개 정답 시:

메시지: “🎉 세계 정복자! 축하합니다!”

쿠폰 이미지: “₩1,000 할인 쿠폰”

버튼: “📥 쿠폰 저장” (이미지 저장 기능)

1~2개 정답 시:

메시지: “😅 X개 정답! 다음 기회에 도전해보세요!”

3개문제 성공시, 축포를 터뜨리고, suc_01.mp3를 내보내라
1~2개 정답시, ggg.png를 좌에서 우로 4초간 내보내고, fail_02.mp3를 내보내라. 



공통 버튼:

“🏠 홈으로” (장면 1로 돌아감)

💡 부가 정보
정답 비교:

대소문자 무시

앞뒤 공백 제거

국가/수도 예시:

프랑스 – 파리

브라질 – 브라질리아

호주 – 캔버라

이 3나라뿐 아니라, 전세계의 모든 나라와 수도를 random으로 문제제출하라. 
객관식 보기를 제시할때의 기준 : 해당나라의 유명한 도시를 4개 제시하라.
예) 호주. 시드니, 캔버라, 멜버른, 브리즈번
일본. 도쿄, 쿄토, 오사카, 나고야
